# Art & Anchal

**Authentic Banarasi Handloom Sarees from Varanasi**

Art & Anchal is a premium e-commerce platform dedicated to showcasing authentic, handwoven Banarasi sarees crafted by legacy weavers of Varanasi. The platform is designed to provide a luxurious shopping experience with high-performance animations, global search, a dynamic admin console, and fully responsive layouts.

## Features
- **Premium Aesthetics**: Glassmorphism UI, scroll-driven animations, high-fidelity responsive imagery, and custom editorial typography.
- **E-commerce Capabilities**: Seamless cart management, curated wishlists, order placement, and seamless checkout flow.
- **Admin Console**: Built-in comprehensive CRM/POS capabilities for managing catalogs, profiles, orders, and leads.
- **Robust Authentication**: Supports standard Email/Password and Google OAuth integrations via Supabase.
- **Performant**: Optimized with dynamic code splitting, eager LCP hints, WebP asset generation, and skeleton loading shimmer states.
- **SEO Optimized**: Pre-configured with meta tags, sitemaps, Open Graph tags, and robotic crawl directives.

## Tech Stack
- **Frontend**: React 18, TypeScript, Tailwind CSS
- **Routing**: Client-side state-based routing
- **Icons**: Lucide React
- **Backend/Database**: Supabase (PostgreSQL)

## Getting Started

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Copy `.env.example` to `.env` and fill in your Supabase credentials:
   ```bash
   VITE_SUPABASE_URL="your-supabase-url"
   VITE_SUPABASE_ANON_KEY="your-supabase-anon-key"
   ```
4. Run the development server:
   ```bash
   npm run dev
   ```
